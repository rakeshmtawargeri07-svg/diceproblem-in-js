Dicee Game 🎲
A simple dice game built using HTML, CSS, and JavaScript.

Project Description:
This project generates random dice numbers for two players whenever the page is refreshed.  
The player with the higher dice number wins the game.

Features:
- Random dice generation using JavaScript
- Dynamic dice image updates
- Winner announcement
- Simple and responsive UI

 Useing
- HTML5
- CSS3
- JavaScript
